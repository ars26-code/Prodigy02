# Stopwatch Web Application

## 📌 Overview

This project is a Stopwatch Web Application developed during my Web Development Internship at Prodigy InfoTech. The application allows users to start, pause, reset, and track elapsed time with an interactive and responsive user interface.

The main objective of this project is to practice JavaScript timing functions and DOM manipulation while creating a clean and user-friendly stopwatch application.

---

## 🚀 Features

* Start stopwatch functionality
* Pause and resume timer
* Reset stopwatch
* Accurate time tracking
* Responsive user interface
* Simple and clean design

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
